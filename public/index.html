Clio Import Instructions
Activities Upload
  1. Download Activities CSV from AR above
  2. Go to app.clio.com → Settings → Imports
  3. Click Add in the top right corner
  4. In the Import dropdown select Activities from CSV
  5. In the User dropdown select Me
  6. Click Choose File and select your Activities CSV
  7. Click Upload File
Expenses Upload
  1. Download Expenses CSV from AR above
  2. Go to app.clio.com → Settings → Imports
  3. Click Add in the top right corner
  4. In the Import dropdown select Activities from CSV (expenses use this option too)
  5. In the User dropdown select Me
  6. Click Choose File and select your Expenses CSV
  7. Click Upload File
